# ACEAdmin
使用ASP.NET Core(2.2) MVC+EF构建的后台管理系统<br><br>
支持MsSQL、MySQL数据库(本例中使用MySQL)<br><br>
支持PC、平板及手机端的操作及展示<br><br>
支持部署至Windows、Linux、Mac OS，验证了在Win7SP1及树莓派(3B+)上均可成功运行<br><br>
示例地址<br>http://cysoft.uicp.net:8081/<br><br>
